Responsive design

What is responsive design?

  • Is a web design approach.

    It is not a technology.

  • Optimal view experience.

    Across devices.

  • Resolution independent.

    Use rem as units.

The virtual viewport

Its metrics

  • 320x480 virtual screen.

    These are CSS pixels.

CSS setup

  • Set font size for <html>.

    html { font: 62.5% "MozTT"; }

  • Default UA font size is 16px.

    62.5% * 16px = 10px

  • Now use rem units.

    rem stands for "root em"

  • Proportional to html font size.

    1rem = 10px; 2rem = 20px;

Support for HDPI devices

  • responsive.css for HDPI

    Add shared/bb/style/responsive.css

  • CSS background size

    background-size: width height;
